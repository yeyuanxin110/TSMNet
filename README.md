# Open-Vocabulary Semantic Segmentation Network Integrating Object Level Label and Scene-Level Semantic Features for Remote Sensing Images

Jinkun Dai, Yuanxin Ye, Peng Tang, Tengfeng Tang, Xianping Ma, Mi Wang

## Abstract
We introduce TSMNet, a novel framework that bridges the gap between visual and linguistic data in remote sensing by leveraging text-guided fusion to achieve robust, open-vocabulary semantic segmentation.
